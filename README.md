# newtext

A simple command-line tool for finding and replacing strings across all text files in the current directory.

## Usage

Run the tool to perform find and replace operations on all text files in your current directory.

## Related Projects

- [rpl](https://rpl.sourceforge.net/) - A text replacement utility originally written for the Debian project that supports recursive directory processing and file type filtering.
- [rpl (Go)](https://github.com/debek/rpl) - A command-line tool written in Go for efficiently replacing strings within the current directory and all subdirectories.
- [sd](https://github.com/chmln/sd) - An intuitive find & replace CLI written in Rust, designed as a more user-friendly alternative to sed.
- [fastmod](https://github.com/facebookincubator/fastmod) - A fast tool from Facebook for large-scale codebase refactors via regex-based find and replace with human oversight.
- [repgrep](https://github.com/acheronfail/repgrep) - An interactive wrapper around ripgrep that provides a simple interface to see replacements in real-time and conditionally replace matches.
- [amber](https://github.com/dalance/amber) - A code search and replace tool with performance benchmarks against various alternatives.
